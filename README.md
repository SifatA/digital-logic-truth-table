# digital-logic-truth-table
This implements the truth tables of basic digital circuits, decoder and multiplexor (not hardcoded).

Truth tables:
  Logic Gates:
    AND
    NOT
    NAND
    OR
    NOR
    EOR(XOR)
  
  Digital cicuits:
    3-8 Decoder
    4-1 Multiplexor (MUX)
    Half Adder
    Full Adder
    3-bit Adder
